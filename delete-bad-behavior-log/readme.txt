Delete Log entries of Bad Behavior plugin

== Description ==

This plugin deletes log entries created by Bad Behavior plugin. Bad Behavior plugin does not have option for deleting log entries.

== Installation ==

1. Upload 'delete-bad-behavior-log' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
